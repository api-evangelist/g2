## APIs

### G2 API
The G2 API provides programmatic access to G2's product, category, and review data. It enables developers to retrieve software product listings, user reviews, category taxonomies, and competitive intelligence data from the world's largest B2B software marketplace. The API supports endpoints for products, categories, reviews, and syndication, and uses token-based authentication.

## Features

| Name | Description |
|------|-------------|
| Product Data Access | Retrieve detailed product listings including descriptions, ratings, and competitive positioning. |
| Review Syndication | Syndicate authenticated user reviews to your own platforms and partner sites. |
| Category Taxonomy | Access G2's comprehensive software category taxonomy for classification and discovery. |
| Competitive Intelligence | Compare products side-by-side using G2's crowd-sourced review data. |
| Buyer Intent Data | Access buyer intent signals showing which companies are researching products in your category. |
