# G2 (g2)

G2 (https://www.g2.com/) is the world's largest and most trusted B2B software and services marketplace, where more than 100 million people annually use peer reviews to make informed software buying decisions. Founded in 2012, G2 hosts millions of verified user reviews across thousands of software products and services categories, providing buyers with authentic insights and sellers with a platform to build reputation, capture buyer intent signals, and grow their business.

## APIs

### G2 API

The G2 API provides programmatic access to G2's product, category, and review data. It enables developers to retrieve software product listings, user reviews, category taxonomies, and competitive intelligence data from the world's largest B2B software marketplace. The API supports endpoints for products, categories, reviews, and syndication, and uses token-based authentication. Partners and developers can use the API to embed G2 review widgets, sync review data, power product discovery features, and integrate G2 buyer intent signals into their own platforms and workflows.
